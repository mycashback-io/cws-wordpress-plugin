Tags: woocommerce, cashback
Requires at least: 5.4
Tested up to: 6.0
Stable tag: 0.0.1
Requires PHP: 5.2.4

Enables customers to recieve cashback on purchase of enabled products on WooCommerce store. Easy to setup and customize cashback  options.

== Description ==

myCashback CWS is an official plugin from the developers of myCashback, the world first Cashback platofrm that offers users 1Day Cashback. 

myCashback CWS is an official WordPress plugin, and will be fully supported and maintained for supported WordPress websites.

At a glance, this plugin adds the following:

* Manage products and cashback options
* Administrators can set cashback validity period and cashback amount.

A valid merchant account is required and can be registered at https://aams.mycashback.io/account/register

== Installation ==

Place the mycashbackbtt folder inside the plugin folder of your wordpress site and activiate through the plugin page on your Administrator dashboard.

== Frequently Asked Questions ==

= Default settings =

When activated this plugin will set cashback validity for product in cart to 24 hours. This can be changed by going to the WooCommerce > Settings > myCashback tab
